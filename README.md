# agrotop-5.0
O AgroTop 5.0 é uma página web desenvolvida para prática de HTML e CSS para o concurso Agrinho 2026. Tendo como objetivo mostrar em uma página web o que foi aprendido nas aulas de Programação.
 O Objetivo da página é ser um guia pratico para auxiliar no desenvolvimento economico sustentavel. 


**🛠️ Tecnologias Utilizadas**
HTML5
CSS3


**🌍 Público-Alvo**
Produtores rurais
Cooperativas agrícolas
Técnicos agrícolas
Engenheiros agrônomos
Instituições de ensino e pesquisa
Empresas do agronegócio


**👨‍💻 Equipe**

Projeto desenvolvido para participar do concurso Agrinho 2026.
